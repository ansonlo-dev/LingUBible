🔧 開發模式測試頁面

✅ 開發模式已成功啟用

您現在可以：

  • 使用任何郵件域名註冊（如 test@gmail.com, user@example.com）
  • 繞過密碼強度檢查（任何密碼都會被接受）
  • 自動通過郵件驗證（無需真實驗證碼）

📧 測試郵件地址

您可以使用以下任何郵件地址進行測試：

test@gmail.com
user@example.com
demo@outlook.com
admin@test.com
student@yahoo.com

或者使用一次性郵件服務：

temp@10minutemail.com
test@tempmail.org
user@guerrillamail.com

🔓 密碼測試

在開發模式下，以下任何密碼都會被接受：

123
password
test
a
任何簡單密碼

⚠️ 重要提醒

這是開發模式設置，僅用於開發和測試。在生產環境中請務必：

  • 將 VITE_DEV_MODE 設置為 false
  • 將 VITE_DEV_BYPASS_PASSWORD 設置為 false
  • 確保只有學校郵件地址可以註冊
  • 啟用完整的密碼強度檢查
