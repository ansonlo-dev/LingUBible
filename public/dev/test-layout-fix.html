🔧 修復說明

問題：Popular Courses 標籤中的「View All」按鈕單獨在一行

解決：改為與標題在同一行，使用 flex items-center justify-between

結果：兩個標籤的佈局現在完全一致

🔍 如何驗證修復

  1. 打開主頁 (http://localhost:8080)
  2. 滾動到 Popular Courses / Top Lecturers 標籤區域
  3. 點擊兩個標籤切換
  4. 確認兩個標籤中的「View All」按鈕都與標題在同一行
  5. 如果沒有變化，請硬刷新瀏覽器 (Ctrl+F5)
